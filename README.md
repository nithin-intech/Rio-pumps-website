# RIO Pumps E-Commerce Website

This project is a dynamic website developed for a pump manufacturing company named **RIO Pumps**.
The website allows customers to view pump products, explore product specifications, send enquiries, and place orders through an interactive online platform.

The project was developed as part of my **Industrial Exposure Training** in the Information Technology department.

---

## 📌 Project Overview

The main goal of this project is to create a professional company website that showcases pump products and allows customers to interact with the company online.

The website includes product categories, product details, enquiry forms, and order placement features similar to a real e-commerce system.

---

## 🛠 Technologies Used

* HTML
* CSS
* PHP
* MySQL
* JavaScript
* XAMPP (Local Development Environment)

---

## ⚙ Key Features

* Company homepage with product highlights
* Product categories (Agricultural, Domestic, Industrial Pumps)
* Detailed product information page
* Add to cart and order placement system
* Customer enquiry form
* Email notification for enquiries
* Download section for product brochures
* Responsive and user-friendly design
